# 我的后台系统

这是一个基于 [Geeker-Admin](https://github.com/HalseySpicy/Geeker-Admin) 进行二次开发的后台管理系统模板。

## ✨ 技术栈

*   **构建工具:** Vite 5.x
*   **前端框架:** Vue 3.4.x
*   **路由:** Vue-Router 4.x
*   **状态管理:** Pinia
*   **UI 库:** Element-Plus
*   **CSS 预处理器:** Sass
*   **代码规范:** ESLint + Prettier

## 🚀 本地开发

1.  **安装依赖**
    ```bash
    pnpm install
    ```

2.  **启动项目**
    ```bash
    pnpm dev
    ```
